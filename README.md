# الكتلة الحجمية للسوائل — PhysicLabDZ

محاكاة تفاعلية لقياس الكتلة الحجمية للسوائل، موجهة لتلاميذ السنة الرابعة متوسط (جيل ثاني).

## الميزات
- بيشر + حنفية + مخبار مدرج + ميزان رقمي
- 4 سوائل: ماء، زيت، كحول، زئبق
- فيزياء واقعية (جاذبية، تصادم، طبقات حسب الكثافة)
- مؤثرات صوتية مخصصة لكل سائل
- تصميم متجاوب: موبايل + سطح المكتب

## الرفع على GitHub Pages
1. ارفع محتوى هذا المجلد كما هو إلى repository
2. فعّل GitHub Pages من Settings → Pages → Branch: main / root
3. الرابط: `https://PhysicLabDZ.github.io/<repo-name>/`

## هيكل الملفات
```
index.html      ← التطبيق الرئيسي
manifest.json   ← إعدادات PWA
sw.js           ← Service Worker (offline)
.nojekyll       ← لـ GitHub Pages

```
